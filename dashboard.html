Linked Mobile Responder App

The SafeZone AI Mobile Responder App is the companion UI for field responders, providing critical tools and real-time updates directly on their mobile devices.

Key Mobile Features

  • **Live Job Alert Screen:** Instant notifications with "Accept" button and incident route map.
  • **Arrival Screen:** Options to "Mark Arrived," "Request Backup," "Mark Resolved" upon reaching location.
  • **Incident Chat:** Real-time communication with citizens (if enabled) and other responders.
  • **Performance Dashboard:** Individual stats for total jobs, average ETA, and user ratings.
  • **Multilingual Toggle:** Supports various South African languages for inclusive communication.
  • **App Notifications:** Push notifications for critical updates and new incidents.
  • **Night Mode:** Optimized display for low-light conditions.
